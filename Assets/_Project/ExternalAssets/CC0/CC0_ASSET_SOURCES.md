# CC0 Online Environment Assets

Imported for Chapter 1 environment replacement.

- Corrugated Steel 007 B, ambientCG, CC0: https://ambientcg.com/view?id=CorrugatedSteel007B
- Painted Plaster 015, ambientCG, CC0: https://ambientcg.com/view?id=PaintedPlaster015
- Planks 039, ambientCG, CC0: https://ambientcg.com/view?id=Planks039
- Qwantani Sunset Pure Sky, Poly Haven, CC0: https://polyhaven.com/a/qwantani_sunset_puresky

The downloaded `.blend` helper files from ambientCG were removed because Unity tried to import them and Blender is not installed. The project keeps the JPG/PBR texture maps used by the materials.
